HTML форми та атрибути

Основні HTML теги для форм

<form>

Тег <form> є контейнером для всіх полів форми. У нього є важливі атрибути:

  • action: вказує URL-адресу для надсилання даних форми.
  • method: визначає метод відправки даних (GET або POST).
  • enctype: використовується для кодування даних (наприклад, multipart/form-data для завантаження файлів).
<form action="/submit" method="post" enctype="multipart/form-data"> <!-- Поля форми --> </form>

<input>

Тег <input> використовується для створення різноманітних полів введення. Має багато атрибутів і типів, які визначають його функціональність.

Використовується для введення даних, які можуть бути текстовими, числовими, датами, паролями, файлами, кнопками тощо.

<label>

Тег <label> асоціює текстову мітку з полем форми, що робить форму більш доступною. Аттрибут "for" прив'язує мітку до конкретного input через його id.

<select>

Тег <select> створює випадаючий список, а <option> визначає окремі варіанти. Атрибути тегу <select>:

  • multiple: дозволяє вибір кількох опцій.
  • size: вказує кількість видимих опцій.
  • disabled: робить список неактивним для вибору.

<option>

<option> визначає окремі варіанти тегу <select>. Атрибути:

  • selected: визначає вибраний за замовчуванням пункт.
  • disabled: робить конкретний варіант неактивним для вибору.

<optgroup>

Тег <optgroup> використовується всередині тега <select> для створення групи опцій. Кожна група може мати свій заголовок (через атрибут label). Це полегшує користувачам навігацію, особливо у великих списках.

  • label: задає заголовок для групи опцій, що буде відображатися як підпис для даної групи.
  • disabled: робить всю групу недоступною для вибору.

<textarea>

Тег <textarea> використовується для введення багаторядкового тексту в HTML-формі. Це ідеальний вибір, коли потрібно ввести великий обсяг тексту, як-от коментарі, відгуки, нотатки тощо. Атрибути:

  • rows: задає кількість видимих рядків у текстовому полі.
  • cols: задає кількість видимих символів в одному рядку.
  • name: задає ім'я для текстового поля, що використовується для передачі даних форми на сервер.
  • placeholder: відображає підказку у текстовому полі, що зникне, коли користувач почне вводити текст.
  • maxlength: обмежує кількість символів, які можна ввести у поле.
  • required: позначає текстове поле обов'язковим для заповнення.
  • readonly: робить текстове поле доступним для перегляду, але забороняє редагування.
  • disabled: повністю деактивує текстове поле, роблячи його недоступним для взаємодії.

<button>

Тег <button> створює інтерактивну кнопку. Атрибути:

  • type: визначає тип кнопки. Можливі значення:
    • submit: відправляє дані форми.
    • reset: скидає значення форми до початкових.
    • button: звичайна кнопка, яка не виконує ніяких дій за замовчуванням. Зазвичай використовується для виклику JavaScript.
  • name: задає ім'я кнопки для передачі даних на сервер. Наприклад, name="myButton" може бути використане для ідентифікації конкретної кнопки у формі.
  • value: задає значення кнопки, яке буде відправлене на сервер, коли кнопка натиснута. Наприклад, value="sendData" передасть на сервер значення "sendData" при натисканні кнопки.
  • disabled: деактивує кнопку, роблячи її недоступною для натискання.
  • autofocus: автоматично надає фокус кнопці при завантаженні сторінки. Наприклад, <button autofocus> встановить фокус на цій кнопці після завантаження сторінки.
  • form: визначає ідентифікатор форми, з якою пов’язана кнопка. Це корисно, коли кнопка розташована поза межами форми. Наприклад, <button form="myForm">, де myForm — це id форми.
  • readonly: робить текстове поле доступним для перегляду, але забороняє редагування.
  • disabled: повністю деактивує текстове поле, роблячи його недоступним для взаємодії.

Типи input та їх атрибути

Загальні атрибути для всіх <input>

  • type — визначає тип введення (наприклад, text, password, email, тощо).
  • name — ім'я, яке використовується для ідентифікації введених даних на сервері.
  • value — початкове значення поля.
  • placeholder — текст, який відображається в полі, коли воно пусте.
  • required — визначає, чи є поле обов'язковим для заповнення.
  • readonly — забороняє редагування поля, але дозволяє переглядати його значення.
  • disabled — повністю деактивує поле, роблячи його недоступним для взаємодії.
  • autocomplete — дозволяє браузеру запам'ятовувати значення, які вводив користувач, і пропонувати їх надалі (значення: on або off).
  • autofocus — автоматично встановлює фокус на полі при завантаженні сторінки.
  • pattern — задає регулярний вираз, який потрібно ввести в поле.
  • minlength і maxlength — задають мінімальну та максимальну кількість символів для введення.
  • form — вказує ID форми, до якої належить поле (корисно, якщо елемент знаходиться поза межами тега <form>).
  • size — визначає ширину поля в символах.

Тип <input> - text

Використовується для введення короткого тексту. Атрибути: placeholder, maxlength, minlength, pattern.

Тип <input> - password

Забезпечує введення пароля (символи приховані). Атрибути: maxlength, minlength, pattern, required.

Тип <input> - email

Перевіряє, що введене значення відповідає формату електронної пошти. Атрибути: placeholder, required.

Тип <input> - number

Призначений для введення числових значень. Атрибути: min, max, step.

Тип <input> - tel

Поле для введення телефонного номера. Не має вбудованої перевірки формату, але може використовуватися разом з pattern. Атрибути: pattern, maxlength.

Тип <input> - url

Призначений для введення URL-адрес. Атрибути: placeholder, required.

Тип <input> - search

Поле для пошукових запитів з можливістю очищення введеного тексту на деяких браузерах. Атрибути: placeholder.

Тип <input> - date

Використовується для вибору дати з календаря. Атрибути: min, max.

Тип <input> - time

Поле для вибору часу. Атрибути: min, max, step.

Тип <input> - datetime-local

Поле для вибору дати та часу без часової зони. Атрибути: min, max, step.

Тип <input> - month

Дозволяє вибирати місяць та рік. Атрибути: min, max.

Тип <input> - week

Дозволяє вибирати тиждень у форматі "рік-тиждень". Атрибути: min, max.

Тип <input> - color

Додає кольоровий вибір.

Тип <input> - checkbox

Дозволяє вибір одного або декількох елементів. Атрибути: checked.

Підписатися на розсилку.

Тип <input> - radio

Дозволяє вибрати один варіант з групи. Атрибути: checked.

Чоловік Жінка

Тип <input> - file

Дозволяє завантажувати файли. Атрибути: accept, multiple.

Тип <input> - hidden

Сховане поле, яке не відображається на сторінці, але передається на сервер.

Тип <input> - submit

Кнопка для відправки форми. Атрибути: value.

Тип <input> - reset

Кнопка для скидання всіх полів форми до початкових значень. Атрибути: value.

Тип <input> - image

Кнопка у вигляді зображення для відправки форми. Атрибути: src, alt, width, height.

Тип <input> - button

Звичайна кнопка, яка не виконує жодних дій за замовчуванням.
