Анімації та переходи в CSS

Переходи в CSS

Переходи (transition) забезпечують плавну зміну значення однієї або декількох CSS-властивостей протягом визначеного часу.

transition: властивість тривалість функція_таймінгу затримка;

  • властивість (property) — CSS-властивість, яку ви хочете змінити (наприклад, color, width, height, background-color тощо). Можна вказати all для зміни всіх властивостей.
  • тривалість (duration) — час, за який має відбутися перехід (наприклад, 0.5s, 300ms).
  • функція_таймінгу (timing-function) — описує, як буде змінюватися значення властивості протягом часу (наприклад, ease, linear, ease-in, ease-out, cubic-bezier()).
  • затримка (delay) — час, через який почнеться перехід (наприклад, 0.2s, 1s).
transition: width 0.5s ease-in-out, height 1s ease-in 0.5s, background-color 0.3s cubic-bezier(0.175, 0.885, 0.32, 1.275) 1.5s;

Анімації в CSS

Анімації (CSS Animations) дозволяють анімувати зміни властивостей за допомогою ключових кадрів (keyframes), контролюючи процес зміни на різних етапах анімації.

animation: назва_анімації тривалість функція_таймінгу затримка кількість_повторень напрямок заповнення стан_гри;

  • назва_анімації (animation-name) — назва, присвоєна анімації у ключових кадрах (@keyframes).
  • тривалість (animation-duration) — час, протягом якого відбувається анімація (наприклад, 2s, 500ms).
  • функція_таймінгу (animation-timing-function) — спосіб зміни властивостей у часі, аналогічний transition-timing-function.
  • затримка (animation-delay) — час до початку анімації.
  • кількість_повторень (animation-iteration-count) — кількість разів, коли анімація повторюється (1, 2, infinite).
  • напрямок (animation-direction) — напрямок анімації (normal, reverse, alternate, alternate-reverse).
  • заповнення (animation-fill-mode) — визначає, чи зберігаються стилі до початку або після завершення анімації (none, forwards, backwards, both).
  • стан_гри (animation-play-state) — визначає, чи буде анімація на паузі або активна (running, paused).
