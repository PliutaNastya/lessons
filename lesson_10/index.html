Позиціонування в CSS: типи, приклади та використання

Позиціонування в CSS

Позиціонування в CSS — це основний інструмент для контролю над тим, як елементи веб-сторінки відображаються на екрані. Завдяки різним типам позиціонування, ми можемо змінювати положення елементів відносно їхніх батьківських елементів, вікна браузера чи інших об'єктів.

CSS дозволяє визначити, як елемент має розташовуватись у документі. Властивість position відповідає за те, де буде знаходитися блок або елемент. Її значення впливають на розміщення елемента, яке може бути динамічним або фіксованим.

element { position: тип; top: значення; right: значення; bottom: значення; left: значення; }

Типи позиціонування

Статичне позиціонування (static)

Це позиціонування за замовчуванням для всіх елементів. Воно не змінює положення елементів відносно їх природного потоку в документі. Статично позиціоновані елементи не реагують на властивості top, right, bottom і left.

Відносне позиціонування (relative)

Елементи з відносним позиціонуванням зміщуються від свого звичайного місця у документі. Вони залишають "порожнє" місце на початковій позиції.

У цьому прикладі елемент зміститься вниз на 10 пікселів і праворуч на 40 пікселів.

Абсолютне позиціонування (absolute)

Елементи з абсолютним позиціонуванням розміщуються відносно найближчого батьківського елемента, який має інше, ніж static, позиціонування. Якщо такого елемента немає, елемент буде позиціонуватися відносно вікна браузера.

Тут елемент буде зміщений на 10 пікселів від верхнього краю свого батьківського елемента і праворуч на 40 пікселів.

Фіксоване позиціонування (fixed)

Фіксовані елементи завжди залишаються на одному місці навіть при прокручуванні сторінки. Вони розміщуються відносно вікна браузера.

Цей блок залишиться у нижньому правому куті незалежно від того, скільки користувач прокручує сторінку.

Липке позиціонування (sticky)

Липке позиціонування поєднує властивості відносного та фіксованого позиціонування. Елемент діє як відносно позиціонований, доки не досягне певної точки прокручування, після чого "прилипає" і веде себе як фіксований.

Цей блок залишиться на своєму місці, доки не буде прокручено до верху сторінки, де він стане фіксованим на 20px нижче верху сторінки.

Порядок накладання (z-index)

z-index визначає порядок накладання елементів, коли вони перекриваються. Елементи з більшим значенням z-index будуть поверх елементів з меншим значенням.

Третій блок буде відображатися поверх інших елементів, тому що в нього вище z-index.

Приклади застосування

Абсолютне позиціонування із центруванням елемента

Цей приклад демонструє, як можна розташувати елемент у центрі свого батьківського елемента.

div { position: absolute; top: calc(50% - 100px); left: calc(50% - 150px); width: 300px; height: 200px; }

Фіксована шапка на сторінці

Нижче наведено приклад створення фіксованої шапки, яка завжди залишається у верхній частині сторінки при прокручуванні. Приклад фіксованої шапки представлено на цій сторінці.

header { position: fixed; top: 0; left: 0; z-index: 50; width: 100%; }
